## PROMPT (ก๊อปตั้งแต่บรรทัดนี้ลงไป)

สร้างเว็บแอป SaaS ชื่อ **VectorCNC** — เครื่องมือที่เปลี่ยน "ภาพถ่าย" ให้เป็น "เส้นตัดเวกเตอร์พร้อม CNC" (SVG/DXF) อัตโนมัติ กลุ่มผู้ใช้คือร้านทำป้าย/งาน CNC ที่รับรูปจากลูกค้าแล้วต้องดราฟท์เป็นไฟล์ตัด UI ภาษาไทย

**อารมณ์ดีไซน์ (สำคัญ):** ธีมมืดแบบ "blueprint / เครื่องมือวิศวกรรมความแม่นยำสูง" — พื้นหลังกรมท่าเข้ม (#0B1220) มีลายกริดจางๆ แบบพิมพ์เขียว, สีเน้นเป็นไล่เฉด cyan→blue (#22D3EE → #3B82F6) เรืองแสงนิดๆ, ฟอนต์ไทยใช้ 'Prompt', ตัวเลข/โค้ดใช้ฟอนต์ monospace 'JetBrains Mono' ดูสะอาด พรีเมียม ทันสมัย (แนว vectorizer.ai แต่หรูกว่า)

**หน้าและองค์ประกอบ:**

1) **Navbar** ค้างบน: โลโก้ VectorCNC (ไอคอนเส้นกราฟหักมุมในสี่เหลี่ยมไล่เฉด) + เมนู (วิธีใช้, ตัวอย่างงาน, ราคา) + ปุ่ม "เข้าสู่ระบบ" และปุ่มเด่น "เริ่มใช้ฟรี"

2) **Hero กลางจอ:** ป้ายเล็ก "ไม่ต้องมี Illustrator · ไม่ต้องดราฟท์มือ", หัวข้อใหญ่ "เปลี่ยนภาพถ่าย เป็น เส้นตัด CNC ใน 10 วินาที" (คำว่า ภาพถ่าย/เส้นตัด CNC เป็นตัวไล่เฉด), คำโปรย, ปุ่มใหญ่ "อัปโหลดภาพเริ่มเลย"

3) **การ์ดแอปจำลอง (workspace)** เหมือนหน้าต่างโปรแกรม (มีจุดแดง-เหลือง-เขียวมุมซ้าย) แบ่ง 3 คอลัมน์:
   - **ซ้าย:** โซนลากรูปมาวาง (drag & drop) + แผงตั้งค่า: สไลเดอร์ "จำนวนสี/เลเยอร์" (2–12), "ความเนียนของเส้น" (ต่ำ/กลาง/สูง/สูงสุด), "Kerf ชดเชยดอกกัด (มม.)", และปุ่มเลือกปลายทาง SVG / DXF / Fusion
   - **กลาง:** ตัวเปรียบเทียบ before/after แบบสไลเดอร์ลากได้ — ด้านซ้ายเป็น "ภาพถ่าย" (เบลอ มีสี), ด้านขวาเป็น "เส้นตัดเวกเตอร์" (เส้น cyan คมบนพื้นเข้มมีกริด) มีเส้นแบ่งเรืองแสงลากได้ ป้ายมุม "ภาพถ่าย" กับ "เส้นตัดเวกเตอร์" ใต้ภาพเขียน "ฟิต Bézier ซับพิกเซล · เนียนกริบ · layout ตรง 100%"
   - **ขวา:** รายการ "เลเยอร์ที่ตรวจพบ" (แต่ละแถวมีสี + ชื่อ + สถานะ ตัด/ข้าม), สถิติคุณภาพ (โหนดเส้น, ความคลาดเคลื่อน px, ขนาดจริง ซม.), และปุ่มดาวน์โหลด "ดาวน์โหลด SVG" (เด่น) + DXF + → Fusion

4) **ส่วน "ทำงานยังไง"** 3 ขั้นตอนเป็นการ์ด: 01 อัปโหลดภาพ (ระบบขยายความละเอียด super-res + ลด noise), 02 AI แปลงเวกเตอร์ (แยกสี/เลเยอร์, ดึงขอบซับพิกเซล, ฟิต Bézier เนียนกริบ + snap เส้นตรง/วงกลม/มุม, OCR อ่านตัวหนังสือวางฟอนต์จริง), 03 ได้ไฟล์ตัด (SVG/DXF พร้อม kerf/tabs/ฟิลเล็ตมุม หรือส่งเข้า Fusion 360 ทำ 3D)

5) **Footer** เรียบๆ

**อินเทอร์แอกชันที่ต้องมี:** สไลเดอร์ before/after ลากได้จริง, สไลเดอร์ตั้งค่าขยับได้, ปุ่ม hover เรืองแสง, responsive (มือถือคอลัมน์เดียว)

ทำให้สวย เนี้ยบ ระดับ production พร้อมนำเสนอนักลงทุนได้

## ขั้นต่อไปหลังได้ UI จาก Claude Design
1. พี่เอ gen แล้วส่ง **URL ของ design** กลับมา
2. อลิซ **import** เข้ามา + ต่อกับ backend จริง:
   - เขียน **FastAPI** หุ้ม `vectorcnc.pipeline.process(image, out_svg, n_colors)`
   - endpoint: `POST /api/vectorize` (รับรูป+ตั้งค่า → คืน SVG/DXF + สถิติ)
   - ต่อปุ่มอัปโหลด/ดาวน์โหลด/สไลเดอร์เข้ากับ API
3. เดโมรันจริง: อัปโหลดภาพ → ได้เส้นตัดจริงจาก vectorcnc
