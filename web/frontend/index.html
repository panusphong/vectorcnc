1. ประโยชน์ของระบบ (ภาพรวม)

  • ลดเวลางานกราฟิก: ลูกค้าส่งมาแค่ภาพ (ถ่าย/AI/โหลดเน็ต) ระบบดราฟท์เป็นเวกเตอร์ .ai ให้ทันที ไม่ต้องลากเส้นเองทีละเส้น
  • ลดข้อผิดพลาดการผลิต: ค่าเผื่อแต่ละชั้น (คิ้ว/อะคริลิค/แผ่นพื้น) คำนวณอัตโนมัติตามมาตรฐาน ไม่ต้องจำ ไม่ต้องบวกลบมือ
  • ประหยัดวัสดุ: Nesting จัดวางชิ้นงานลงแผ่นให้แน่นที่สุด รู้จำนวนแผ่น/เปอร์เซ็นต์การใช้วัสดุก่อนตัดจริง
  • สื่อสารชัดเจน: ภาพโครงสร้าง 3 มิติมีมิติ + เส้นจับขนาดต่อชิ้น ให้ช่างและลูกค้าเข้าใจตรงกัน
  • มาตรฐานเดียวทั้งทีม: ทุกคนใช้ค่าเผื่อ/วิธีเดียวกัน งานออกมาสม่ำเสมอ
  • เก็บประวัติงาน: บันทึก Job (ใคร/เมื่อไหร่/ลูกค้า/แบบ) ค้นย้อนหลังได้
  • ออกเอกสารสั่งผลิต: สร้าง 🧾 ใบสั่งผลิต (3มิติ + โครงเหล็ก + LED + Cut Layers + BOM) ให้ลูกค้าเซ็นอนุมัติ ในคลิกเดียว
  • รองรับงานหลากหลาย: ตั้งแต่กล่องไฟ/ตัวอักษร ไปจนถึง โครงเหล็กแขวน และ นีออนเฟล็กซ์

2. ประโยชน์แยกตามฝ่าย

ฝ่ายใช้ทำอะไรได้ประโยชน์
กราฟิกดราฟท์ภาพ→.ai, แปลงเส้นตัด, ทำชุดชั้นตัดประหยัดเวลาลากแบบ 60–80% มีเวกเตอร์ตั้งต้นเปิดใน Illustrator ต่อได้ทันที
เซลล์ขนาดป้าย+จำลองผนัง, เสนอราคา, บันทึก Jobโชว์ลูกค้าเห็นภาพจริงบนผนัง คิดราคาเร็ว ปิดงานไว
ประเมินราคาวัดขนาดจริง, W×H ต่อชิ้น, เส้นรอบรูปรู้พื้นที่/ความยาวขอบแม่นยำ คิดต้นทุนวัสดุถูกต้อง
ฝ่ายผลิต / ช่างตัด CNCรับไฟล์ DXF เส้นตัดคม, Nesting, ชุดชั้นตัดแยก layerไฟล์พร้อมเข้าเครื่อง (CypCut/fiber/เลเซอร์) เส้นเนียน ไม่ต้องแก้
ช่างประกอบ / หน้างานดูภาพ 3 มิติมีมิติ, ตารางชั้น, ยกขอบ (พับ)รู้ว่าชิ้นไหนคือชั้นไหน ประกอบถูก ไม่งง
ผู้จัดการ / เจ้าของประวัติ Job, มาตรฐานค่าเผื่อ, ควบคุมของเสียงานสม่ำเสมอ ตรวจสอบย้อนหลังได้ ลดของเสีย/แก้งาน

2. ระบบแตกทุก object ออกมาให้เห็นเป็นการ์ด พร้อมป้ายกำกับ: เวกเตอร์ ✓ = ใช้ได้เลย · ภาพ = ต้องดราฟท์ต่อ · ข้อความ = บอกชื่อฟอนต์ให้ด้วย
3. กดการ์ดที่ต้องการ → ดาวน์โหลดเป็น .ai ทันที
จุดสำคัญ: ถ้าชิ้นนั้นเป็นเวกเตอร์อยู่แล้ว ระบบจะตัดออกมาเป็นเวกเตอร์ต้นฉบับ 100% — ไม่ trace ใหม่ ไม่เสียความคม ไฟล์เล็ก (หลัก KB) และเปิดแก้ path ได้ใน Illustrator
ถ้าโลโก้ในไฟล์เป็น ภาพ raster (ฝังในเป็นรูป) จะได้ .png ออกมา ต้องส่งเข้า ดราฟท์ .ai ที่หน้าหลักต่อ เพื่อแปลงเป็นเวกเตอร์

✅ 2) ตรวจว่าผลิตได้ — ด่านกรองก่อนรับปากลูกค้า

ใช้เมื่อ: ลูกค้าส่งภาพจาก AI ตัวอื่น (Midjourney / ChatGPT / Gemini) หรือภาพที่ดูสวยแต่ไม่รู้ว่าตัดได้ไหม

ระบบตรวจ 7 ข้อ:

ตรวจทำไมสำคัญ
🔴 เส้น/แขนบางเกินไปวัด "จุดแคบที่สุด" จริง ๆ (ไม่ใช่แค่ชิ้นเล็ก) — บางเกินตัดแล้วหัก/บิด
🟠 ชิ้นเล็กเกินไปหลุดร่วงตอนตัด เก็บงานยาก หาย
🟣 ช่อง/ไส้ในเล็กเกินไปดอกกัดเข้าไม่ถึง และคิ้วจะปิดทับ
🔵 ชิ้นชิดกันเกินไปตัดแล้วเชื่อมติดกัน
จุดบนเส้นถี่เกินเครื่องดัดจะกรีดพับถี่ → ขอบเป็นเหลี่ยม ไม่เนียน
เส้นซ้อน/ตัดกันเองCAM ตัดผิด
ชิ้นย่อยเยอะผิดปกติบอกว่าเป็นภาพถ่าย/ไล่เฉด — ป้ายทำไม่ได้
ผลลัพธ์: คะแนน 0–100 + ผลิตได้ ✓ ต้องแก้ก่อน ผลิตไม่ได้ ✕ + ภาพวงแดงชี้จุดที่มีปัญหา + บอกว่าต้อง "หนาขึ้นอีกกี่ มม." หรือ "ขยายงานกี่ %"
ปุ่ม 🔧 แก้อัตโนมัติ: ระบบซ่อมให้เอง — เพิ่มความหนาให้ถึงขั้นต่ำ · อุดช่องเล็ก · ลบชิ้นจิ๋ว · ลดจุดบนเส้น · ซ่อมเส้นซ้อน → ดาวน์โหลด .ai ที่ผลิตได้จริง
เกณฑ์ต่อวัสดุ (ปรับได้): อะคริลิค 4 มม. · ซิ้งค์/โลหะ 3 มม. · พลาสวูด 6 มม. · สแตนเลส 2.5 มม. · กล่องไฟ 8 มม.
ใช้ก่อนเสนอราคา — ไม่ใช่หลังกราฟิกทำไปครึ่งวันแล้ว นี่คือจุดที่ประหยัดเวลามากที่สุด

💡 3) AI Concept — ลูกค้าไม่มี idea / ไม่มีชื่อ / ไม่มีโลโก้

ใช้เมื่อ: ลูกค้ามาแบบว่างเปล่า เซลล์ไม่มีอะไรให้ดู งานค้างอยู่ตรงนั้น

1. ขอชื่อร้าน: เลือกประเภทธุรกิจ + โทน → กด 🎲 ได้ชื่อ 10 ชื่อ พร้อมเหตุผล (กดชื่อที่ชอบ → เด้งไปช่อง "ชื่อบนป้าย" ให้เลย)
2. สร้างโลโก้: ใส่ชื่อ + ความสูงตัวอักษร → ได้แบบหลายสิบแบบ (สไตล์ฟอนต์ × เลย์เอาต์ 7 แบบ: ตัวอักษรล้วน / คิ้วรอบ / เส้นใต้ / แผ่นป้าย / ตราวงกลม / กรอบ / ชื่อ+คำบรรยาย)
3. ทุกแบบตรวจผลิตได้แล้วในตัว — เรียงตามคะแนน กดแบบที่ชอบ → ได้ .ai ทันที
จุดสำคัญ: โลโก้ที่ได้เป็น เวกเตอร์จริงจากฟอนต์ ไม่ใช่ภาพ AI — คมกริบ ตัดได้จริง ส่งเข้าชุดชั้นตัด / Nesting ต่อได้เลย (ต่างจากภาพ Midjourney ที่ผลิตไม่ได้)

📝 4) Sales Brief — ตัดเวลา "ถามกลับ"

ใช้เมื่อ: เซลล์รับงาน — กรอกที่หน้าลูกค้าเลย

14 ช่องมาตรฐาน: ชื่อลูกค้า · ข้อความบนป้าย · ประเภทป้าย 1–7 · กว้าง × สูง · ความหนายกขอบ · วัสดุ · จำนวน · ติดตั้งที่ไหน · ไฟฟ้าถึงหรือยัง · กำหนดส่ง · งบ · ไฟล์ต้นแบบ · หมายเหตุ

กด "ตรวจความครบ" → ระบบบอก % ความครบ และ ยังขาดอะไรบ้าง + สร้าง ใบบรีฟ ดาวน์โหลดส่งกราฟิกได้เลย
ช่องที่มี * = จำเป็น — ถ้าไม่ครบ กราฟิกจะต้องเดา และงานจะวนแก้

4. คู่มือใช้งานทีละฟีเจอร์

4.1 แปลงภาพเป็นเส้นตัด (หน้าหลัก)

ใช้เมื่อ: มีภาพ/โลโก้ ต้องการไฟล์ตัด CNC/เลเซอร์

1) ลากไฟล์มาวาง (JPG/PNG/PSD หรือ AI/PDF/SVG/EPS)
2) เลือกวัสดุ/เครื่อง (Fiber·สแตนเลส / Fiber·ซิงค์ / CO2·อะคริลิค / Router·ดอกกัด)
3) เลือกโหมด (อัตโนมัติ/ตัดชิ้น/เส้น) + จำนวนสี
4) กำหนดขนาด (กว้าง/สูง/สูงตัวอักษร มม.) — สำคัญมากกับภาพ JPG
5) ตั้ง Kerf (ชดเชยลำ/ดอก) + ดอกกัด/ปัดมุม ถ้าต้องการ
6) กด "⚡ แปลงเป็นเส้นตัด" → ดาวน์โหลด SVG/DXF
💡 DXF → CypCut/fiber · SVG → LightBurn · Fusion 360: Insert DXF · ดูไฟล์ให้เนียนสุดใน LibreCAD/Inkscape/LightBurn

4.2 ดราฟท์ภาพ → .ai (ให้กราฟิกทำต่อ)

ใช้เมื่อ: ลูกค้าส่งมาแค่ภาพ กราฟิกต้องได้เวกเตอร์ตั้งต้นเปิดใน Illustrator

1) อัปโหลดภาพในหน้าหลัก → ใส่ขนาดจริง
2) เลือกโหมด: อัตโนมัติ / ลายเส้นคม (ขาว-ดำ) / หลายสี
3) กด "🎨 ดาวน์โหลด .ai (ให้กราฟิก)"

ได้: ไฟล์ .ai เวกเตอร์ (path แยกชิ้น แก้ได้) ขนาดจริงตามงาน

💡 ภาพลายเส้น/โลโก้ → auto เลือกโหมดคมกริบให้เอง · ภาพสีเรียบ → แยกสีเป็น path คนละชั้น

4.3 ชุดชั้นตัด + 3 มิติ (ตามแบบป้าย)

ใช้เมื่อ: ทำป้ายตัวอักษร/กล่องไฟที่มีโครงสร้างหลายชั้น (ไม่ใช่แผ่นแบน)

ทำได้ 2 ทาง: (ก) ในหน้าหลัก เลือก "ประเภทป้าย" ข้างช่องขนาด → กด "🏭 ชุดชั้นตัด + 3 มิติ" · (ข) เมนู "🏭 ชุดชั้นตัด (ตามแบบป้าย)" ด้านบน

1) อัปโหลดตัวอักษร/โลโก้ + กำหนดขนาด (มม.)
2) เลือกประเภทป้าย (กล่องไฟ / ตัวอักษร / โครงแขวน / นีออน ฯลฯ)
3) กด "🏭 ชุดชั้นตัด + 3 มิติ"

ได้ (แสดงในพื้นที่กลางจอ · แบ่ง 2 ช่อง):

  • 🧊 3D Structure — ภาพ 3 มิติมีมิติ (สูง/กว้าง/ลึก) เห็นผนังข้าง (Return) + คิ้วเจาะโบ๋
  • 📐 Cut Layers — สเปคจับขนาดต่อชิ้น + ตาราง layer
  • เส้นตัดแยกชั้นตามวัสดุ: คิ้ว(เจาะโบ๋)/อะคริลิค(−0.25)/แผ่นพื้น(+0.1) · ยกขอบ = แถบพับ (ยาว = เส้นรอบรูป × สูง)
  • ชื่อภาษาอังกฤษ ในภาพ/DXF: Trim Face (Kim) · Acrylic Face · Back Plate · Return … (มีไทยกำกับในตาราง)
  • ปุ่ม ⬇ DXF (แยก layer/วัสดุ) + "← กลับไปดูเส้นตัด"
⚠️ ภาพ JPG/PNG ไม่มีขนาดกำกับในไฟล์ (ต่างจาก .ai) — ต้องกำหนดขนาดก่อนเสมอ ไม่งั้นชิ้นงานจะผิดขนาด (ระบบจะเตือนถ้ายังไม่ใส่)

4.4 Nesting — จัดวาง 1 ไฟล์ลงแผ่น

ใช้เมื่อ: รู้ชิ้นงานแล้ว ต้องการจัดลงแผ่นวัตถุดิบให้ประหยัด

1) เลือกโหมด: แยกชิ้นย่อย (ตัดทุกเส้น แพคชิด) หรือ ทั้งป้าย (เฉพาะกรอบนอก)
2) เลือกแผ่น (อะคริลิค 1220×2440 / โลหะ 1200×2400…) หรือกำหนดเอง
3) กำหนด กว้าง×สูงชิ้น (🔒 ล็อกสัดส่วน) + จำนวน + เว้นระยะ
4) กด "▦ จัดวาง" → เห็นจำนวนแผ่น/ใช้วัสดุ% + เส้นจับขนาดต่อชิ้น → โหลด DXF

4.5 จัดวางหลายไฟล์รวมแผ่น

ใช้เมื่อ: มีหลายงาน/หลายไฟล์ อยากตัดรวมแผ่นเดียวประหยัดสุด

1) ลากไฟล์ (สูงสุด 10) → แต่ละไฟล์ได้รหัส A/B/C… คนละสี
2) ตั้งค่าต่อไฟล์: ขนาด (🔒), จำนวน, โหมด (แยกชิ้น/ทั้งป้าย)
3) เลือกแผ่น + เว้นระยะ + ช่องเส้นกั้น → กด "จัดวางรวม"

ได้: จัดวางแบบแยกโซนต่อไฟล์ + เส้นกั้น + ป้ายรหัส (เก็บชิ้นออกจากเครื่องง่าย) · DXF รวมทุกแผ่น / แยก DXF รายไฟล์ / PDF พรีวิว

🏭 โหมด "ตัดตามโครงสร้างป้าย (แยกชั้นวัสดุ)"

กดปุ่ม "🏭 ตัดตามโครงสร้างป้าย" ในหน้าให้เป็น "เปิด" → แต่ละไฟล์เลือก ประเภทป้าย (1–7) ได้เอง

ระบบจะแตกทุกไฟล์เป็นชั้น (คิ้ว/อะคริลิค/แผ่นพื้น/ยกขอบ) แล้ว รวมชิ้นตามวัสดุข้ามทุกไฟล์
จัดวาง แยกแผ่นต่อวัสดุ (คิ้วรวมแผ่น · อะคริลิครวมแผ่น · แผ่นพื้นรวมแผ่น · ยกขอบรวมแผ่น) — ถูกต้องตามการผลิต (คนละวัสดุคนละแผ่น/เครื่อง)

ได้: ผลลัพธ์แยกเป็นหมวดวัสดุ (จำนวนชิ้น/แผ่น/ใช้วัสดุ%) + ดาวน์โหลด DXF แยกต่อวัสดุ เอาเข้าเครื่องตัดทีละวัสดุ

4.6 ขนาดป้าย + จำลองผนัง

ใช้เมื่อ: เซลล์ต้องการวัดขนาด เสนอราคา และโชว์ลูกค้า

1) วัดขนาดจริงจากภาพ
2) วางชิ้นงานบนภาพผนัง (ปรับสเกล/หมุน) — มีกล่องราคา (font Prompt) + กว้าง×สูง
3) ดู "🧊 3 มิติ · หมุนรอบตัว" (แผ่นหนา/ตัวนูน/ป้ายไฟ)
4) Export JPG/PDF (มีขนาด+ระยะจากผนัง) · 💾 บันทึกงาน / 🔎 ค้นหางาน
💡 ส่งขนาดเข้าแอปประเมินราคาอัตโนมัติ / ใส่ลง BOM Check Sheet ได้

4.7 BOM Check Sheet

ใช้เมื่อ: ต้องการใบตรวจรายการวัสดุ/ชิ้นงานก่อนผลิต — โหลดข้อมูลจากใบสั่งผลิต แล้วตรวจครบถ้วน

4.8 โครงเหล็กยึดตัวอักษร + มุมมองด้านหลัง ใหม่

ใช้เมื่อ: ทำตัวอักษรโลหะ/อะคริลิคที่ต้อง "แขวนลอย" จากผนัง/เพดานด้วยโครงเหล็ก (ประเภท "อักษรยกขอบไฟออกหน้า + โครงแขวน")

1) เลือกประเภทป้ายนี้ — หรือ พิมพ์ข้อความได้เลย (ไม่ต้องอัปโหลดภาพ)
2) เปิดแผง "ปรับโครงเหล็ก" ตั้งค่า: ระยะแขนจากขอบซ้าย-ขวา · ความสูงแขน · ระยะคานบน-ล่าง (สูงเฟรม) · standoff (ระยะห่างจากหลังป้าย) · ระยะรูสายไฟ
3) กด "🏭 ชุดชั้นตัด + 3 มิติ" → ได้ทั้งภาพ 3 มิติหน้า + มุมมองด้านหลังโชว์โครง + รูเจาะ

โครงเหล็ก = "คานคู่แนวนอน" — คานบน + คานล่างพาดกลางตัวอักษร ปิดหัวท้ายซ้าย-ขวาเป็นกรอบ + 2 แขน ที่ปลายซ้าย-ขวายื่นขึ้นยึดเพดาน

  • รูน็อต Ø3 (น้ำเงิน): เจาะบนตัวอักษรตรงแนวคานบน + คานล่าง (ยึดอักษรเข้ากับโครง)
  • รูสายไฟ Ø5 (แดง): จุดร้อยสายไฟ หลบแนวโครง 1 ซม.
  • จับระยะครบ: เฟรมกว้าง × สูง · ความสูงแขน · ระยะแขนจากขอบ · ระยะขอบบน/ล่างอักษรถึงคาน
💡 รูน็อตจะขยับตามตำแหน่งคานโครงเหล็กให้อัตโนมัติ — เลื่อนคาน/ปรับสูงเฟรม รูจะย้ายตามให้อยู่บนตัวอักษรเสมอ ไม่หลุดขอบ

4.9 นีออนเฟล็กซ์ (LED Neon Flex) ใหม่

ใช้เมื่อ: ทำป้าย/ตัวอักษรแนวนีออน เส้นไฟยืดหยุ่นบนแผ่นอะคริลิคใส (ประเภท "นีออนเฟล็กซ์")

1) เลือกประเภทป้ายนี้ → อัปโหลดภาพลายเส้น/โลโก้
2) เลือก สีนีออน 12 สี + รูปแบบเส้น: เส้นเดี่ยว (แกนกลาง) หรือ เส้นคู่ (ขอบสองข้าง)
3) กด "🏭 ชุดชั้นตัด + 3 มิติ"

ได้ชั้นงานครบ:

  • เส้นนีออน — แปลงลายเส้นในภาพลูกค้าเป็นแนวนีออนสีที่เลือก
  • อะคริลิคใสรองหลัง 8 มม. — ล้อมทรง + เผื่อขอบเข้ามา ~3 ซม. พร้อมจุดเจาะหมุดยึดผนัง + จุดสายไฟออก
  • ไฟล์เซาะร่อง CNC (ลึก 4 มม.) — แยกเลเยอร์สำหรับเครื่องเซาะร่องฝังเส้นนีออน

4.10 🧾 ใบสั่งผลิต / แบบยืนยันลูกค้า (Job Sheet) ใหม่

ใช้เมื่อ: สรุปงานเป็นเอกสาร 1 ใบ ส่งโรงงานผลิต + ให้ลูกค้าเซ็นอนุมัติ

ในหน้า "ชุดชั้นตัด + 3 มิติ" → กรอกแผง "📋 รายละเอียดสั่งผลิต" (วัสดุหลัก · ชนิดไฟ · สายไฟ · งานพิมพ์ · กำหนดส่ง) → กด "🧾 ใบสั่งผลิต"

เอกสารมีครบ:

  • ภาพ 3 มิติ + วัสดุหลัก · โครงเหล็กแขวน (มุมหลัง + รูเจาะ) · การวางไฟ LED + คำนวณกำลังไฟ/หม้อแปลง/สายไฟ
  • งานพิมพ์ (artwork) · ภาพจัดเรียงแผ่น (Nesting)
  • ชิ้นตัดแยกชั้น (Cut Layers) — ทุก layer พร้อม allowance (+ ขยาย / − หด) + ขนาดตัด W×H ต่อชิ้น
  • ตาราง BOM (วัสดุ/สเปค) · ช่องเซ็นอนุมัติ 3 ฝ่าย · กำหนดส่งมอบ
💡 รายละเอียดชุดเดียวกันยังแสดงกลางจอในหน้าออกแบบด้วย ดูก่อนพิมพ์ได้ · กด 🖨️ ในเอกสารเพื่อบันทึกเป็น PDF · หน้านี้มีปุ่ม 💾 บันทึกงาน และ 🔎 ค้นหางานที่บันทึกไว้

5. ความรู้ประเภทป้าย + ค่าเผื่อ

ป้าย/ตัวอักษรเป็นงาน "ประกอบหลายชั้น" แต่ละชั้นตัดคนละวัสดุ และตัดขนาดต่างกันเล็กน้อย (ค่าเผื่อ/offset) จาก "ไซซ์เต็ม" เพื่อประกอบพอดี

#รูปแบบชั้นตัด (ค่าเผื่อ)ยกขอบ (ผนัง·พับ)
1ไฟออกหน้า มีคิ้วคิ้ว (เจาะโบ๋·เต็ม) · อะคริลิค −0.25 · แผ่นพื้น +0.15 ซม. + ใน 2 ซม.
2ไฟออกหน้า ไม่มีคิ้วหน้าอะคริลิค +0.1 · ไส้อะคริลิคใส −0.15 · แผ่นพื้น เต็ม5 ซม.
3ตัวอักษรไฟออกรอบหน้าอะคริลิค เต็ม · แผ่นพื้น +0.1 · ข้างอะคริลิค แสงออกรอบตัวอักษรใน 2 ซม. + อะคริลิค 7 ซม.
4กล่องไฟฉลุหน้าคิ้ว · หน้าฉลุตัวอักษร −0.25 · แผ่นพื้น +0.15 ซม. + ใน 2 ซม.
6งานยกขอบซิ้งค์ เต็ม2.5 ซม. + ขากลางยกลอย
7งานยกขอบ มีไส้หน้าซิ้งค์ เต็ม · ไส้พลาสวูด −0.162.5 ซม.
16อักษรยกขอบไฟออกหน้า + โครงแขวนหน้า/คิ้ว/แผ่นพื้น + โครงเหล็กคานคู่ (รูน็อต Ø3 / สายไฟ Ø5)โครงแขวน 2 แขน · standoff 5 ซม.
17นีออนเฟล็กซ์เส้นนีออน 12 สี · อะคริลิคใส 8 มม. (+3 ซม.) · ร่องเซาะ CNC 4 มม.เจาะหมุดยึดผนัง + จุดสายไฟออก
18กล่องไฟอะคริลิค ไฟออกรอบกล่องสี่เหลี่ยม · หน้าอะคริลิคขาวพิมพ์ (โลโก้/พิมพ์ Text ได้) · ขอบเรืองแสงรอบเลือกแขวนเพดาน/ติดผนัง · กว้าง+ลึกปรับได้

หลักค่าเผื่อ: ไซซ์เต็ม (0) = เส้นออกแบบจริง (หน้า/คิ้ว/ซิ้งค์) · +0.1 = ขยาย (แผ่นพื้น) · −0.25 = หด (อะคริลิคตู้ไฟ) · −0.15/−0.16 = หด (ไส้ใน) · คิ้ว = เจาะโบ๋ (กรอบกลวงตามความกว้างคิ้ว) · ยกขอบ = ตัดเป็นแถบยาว=เส้นรอบรูป แล้วพับ

6. ตัวอย่าง Workflow (รับงาน → ตัด)

  1. เซลล์รับงาน: ลูกค้าส่งภาพโลโก้ → "ขนาดป้าย+จำลองผนัง" วางบนผนัง เสนอราคา บันทึก Job
  2. กราฟิก: อัปโหลดภาพ → "🎨 .ai" ได้เวกเตอร์ → เก็บรายละเอียดใน Illustrator
  3. เตรียมผลิต: เลือกประเภทป้าย (เช่น 1 Front-lit·with Trim) → ใส่ขนาด → "🏭 ชุดชั้นตัด+3มิติ" → เห็นภาพ 3 มิติ + สเปคชื่ออังกฤษกลางจอ + เส้นตัดแยกชั้น + ยกขอบ
  4. จัดแผ่น: "จัดวางหลายไฟล์" → เปิดโหมด "ตัดตามโครงสร้าง (แยกวัสดุ)" → DXF แยกต่อวัสดุ (คิ้ว/อะคริลิค/แผ่นพื้น/ยกขอบ) พร้อมตัด
  5. ช่างตัด: เปิด DXF ในเครื่องตัด (เลือก layer วัสดุ) → ตัด
  6. ช่างประกอบ: ดูภาพ 3 มิติ + ตารางชั้น → ประกอบตามชั้น (คิ้ว/อะคริลิค/แผ่นพื้น/พับยกขอบ)

7. เคล็ดลับ & ข้อควรระวัง

  • สำคัญ กำหนดขนาดก่อนสร้างไฟล์เสมอ — ภาพ JPG/PNG ไม่มีขนาดกำกับเหมือน .ai
  • ล็อกสัดส่วน 🔒 ไว้ กันภาพยืด/บิด (ปลดล็อก 🔓 เมื่ออยากยืด/หดอิสระ)
  • เปิด DXF ในโปรแกรมพื้นฐาน (Windows preview) เส้นโค้งจะดูหยาบเอง — ให้ดูใน LibreCAD/LightBurn/Inkscape จะเห็นเส้นเนียนจริง
  • ตั้ง Kerf ให้ตรงกับลำ/ดอกจริง เพื่อให้ชิ้นประกอบพอดี
  • ยกขอบ (แผ่นข้าง) = ตัดเป็นแถบแล้วพับ ความยาวใช้เส้นรอบรูป — ตรวจก่อนดัด

8. คำถามที่พบบ่อย (FAQ)

Q: เส้น DXF ดูยึกยัก?
A: ตัวไฟล์เนียน แต่โปรแกรมดูพื้นฐานจะแตกเส้นโค้งหยาบเอง — เปิดใน LibreCAD/LightBurn/Inkscape

Q: Nesting วางไม่ได้/ว่างเปล่า?
A: เช็กว่าขนาดชิ้นไม่ใหญ่เกินแผ่น และเว้นระยะไม่มากเกินไป

Q: ทำไมต้องใส่ขนาดทุกครั้ง?
A: ภาพ JPG/PNG ไม่มีหน่วยจริงในไฟล์ ระบบต้องรู้ว่าจะให้กว้าง/สูงเท่าไหร่จึงจะสเกลถูก

Q: คิ้วทำไมต้องเจาะโบ๋?
A: คิ้วคือขอบลิปด้านหน้า (กว้าง ~1 ซม.) เจาะกลางทะลุให้แสง/อะคริลิคโผล่ — ไม่ใช่แผ่นตัน
